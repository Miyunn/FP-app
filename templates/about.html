About


The goal of this application is to lay the groundwork for handwritten character recognition in Sinhala. Many diverse sectors use and demand handwritten character recognition. In order to read addresses and sort mail using an automated method, the US Postal Service uses handwritten recognition. This is only one of many applications for handwritten character recognition.

For such systems to be a reality there needs to be a sufficient base. This application strives to collect the data that is needed while providing the community a platform to practice their Sinhala handwriting. The source code datasets and other necessary information of this project can be accessed through the GitHub repository.
